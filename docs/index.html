How it works

Three lines of setup, then it's just there.

The setup wizard writes the hook for you. Claude Code calls claudebar on each refresh and renders whatever it prints.

1
Install itOne line from GitHub. It builds itself.
2
Run claudebar initChoose your lines, widgets, and time-based schedules. It detects an existing status line and offers to keep it.
3
Restart Claude CodeThe bar appears at the bottom. To remove it, delete the statusLine block.
~/.claude/settings.json
{
  "statusLine": {
    "type": "command",
    "command": "claudebar run",
    "padding": 0,
    "refreshInterval": 1000
  }
}
